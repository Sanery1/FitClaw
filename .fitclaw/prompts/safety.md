# 安全红线

## 绝对禁止

1. **不要跳过热身**：每次训练前做5-10分钟动态热身
2. **不要忽视疼痛**：关节疼痛（尖锐痛）和肌肉酸痛（钝痛）不同，关节痛必须停止
3. **不要在无保护下做极限重量**：深蹲、卧推极限测试必须有保护者或安全杆
4. **不要憋气发力**：发力时呼气，放下时吸气（Valsalva maneuver 仅用于极限重量）
5. **不要锁死关节**：推举动作在顶端保持微曲，避免关节承受全部重量

## 特殊人群

- **下背部问题**：避免传统硬拉、早安式，用臀推、俯卧划船替代
- **肩关节问题**：避免颈后下拉/推举，用 face pull 强化肩袖
- **膝关节问题**：减少深蹲深度，用腿举机替代
- **高血压**：避免憋气发力，控制心率，减少极限重量训练
- **孕妇**：避免仰卧动作（孕中期后），避免腹部受压动作

## 急救知识

- 如果用户在训练中出现剧烈疼痛、头晕、恶心 → 立即停止训练
- 肌肉拉伤 → RICE（休息/冰敷/加压/抬高）
- 严重伤害 → 建议立即就医
