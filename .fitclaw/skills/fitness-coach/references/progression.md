# 进阶触发条件与策略

## 双重进阶法（Double Progression）

增肌训练的标准进阶方法：

1. 设定目标次数范围（如8-12 reps）
2. 先用较轻重量做到目标范围的顶部（12次）
3. 达成后增加重量（2.5-5%），此时只能做目标范围底部（8次）
4. 在新重量下逐步增加次数直到再次达到顶部
5. 重复循环

示例：
- 第1周：60kg 卧推 4×8（最低次数）
- 第2周：60kg 卧推 4×10
- 第3周：60kg 卧推 4×12（达到目标顶部）
- 第4周：62.5kg 卧推 4×8（增加重量，回到底部）

## 线性进阶（Linear Progression）

力量和初学者适用：
- 每次训练/每周增加相同重量（如+2.5kg）
- 深蹲/硬拉可以每次+2.5-5kg
- 上肢动作每次+1.25-2.5kg
- 当无法完成目标次数时，重置到80%重新开始

## 进阶触发条件汇总

| 条件 | 动作 |
|------|------|
| 连续2周在目标次数范围顶部完成 | 增加重量2.5-5% |
| 连续2周无法达到次数范围底部 | 降低重量5-10% |
| RPE持续低于目标范围 | 增加重量 |
| RPE持续高于目标范围 | 降低重量 |
| 出现关节疼痛 | 停止进阶，检查动作形式 |
| 平台期超过4周 | 考虑换动作变式或调整训练量 |

## 何时使用 deload（减载周）

- 每6-8周安排一次减载周
- 重量降低到正常的60-70%
- 维持组数和次数不变
- 减载周后通常有超量补偿效果
